📄 Resume Website Project

🔹 Project Overview

This project is a simple Resume Website created using only HTML as part of my internship at Hex Softwares.
It represents my professional resume in a clean and structured web format.

🎯 Objective

To build a resume website using basic HTML
To understand structuring content using HTML tags
To present personal and professional details in a web format

🛠️ Technologies Used:
HTML5

📑 Features:

Header section with name and contact details
Professional Summary
Education details
Projects section
Certifications
Skills and Languages
Clean and readable layout

🚀 How to Run the Project:
Download or clone the repository
Open the folder
Open index.html in any browser
